# Simple CLI Calculator

A robust, user-friendly command-line calculator built with Python. This application handles basic arithmetic operations while gracefully managing invalid user inputs and edge cases like division by zero.

## Features

- **Core Arithmetic:** Supports Addition, Subtraction, Multiplication, and Division.
- **Input Validation:** Prevents crashes by catching non-numeric inputs and prompting the user to try again.
- **Safety Guards:** Gracefully handles division-by-zero errors without terminating the program.
- **Clean Interface:** Formats results dynamically (e.g., hiding unnecessary decimals for whole numbers) and loops continuously until you choose to exit.

---

## Getting Started

### Prerequisites

You only need **Python 3.6 or higher** installed on your machine. No external dependencies or libraries are required.
